# 票务内测预发联调检查表

## 目的

这份检查表把联合 runbook 落成一张可以逐项勾选的预发签收单，适用于 `1 场真实演出` 的正式放量前检查。

在任何公开售卖前都先用它过一遍。只要还有必填项没过，或者没有明确责任人和补救方案，就不能判定为“可开卖”。

## 第七部分：退款联调

## 第八部分：规则核验

- [ ] 电子票文案显示 `演出开始前三天` 规则。

- [ ] 订单详情明确说明“下单后、出票前不能修改观演人信息”。
- [ ] `USER_IDENTITY_ERROR` 且 `daysBeforeStart <= 3` 时，退款计算会扣除 `20%` 服务费。
- [ ] 非身份信息错误原因不会误扣这笔服务费。

- [ ] 正常路径：登录 -> 草稿单 -> 支付 -> 出票。
